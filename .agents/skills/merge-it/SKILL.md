# Merge It

This workflow is retired.

## Required Behavior

- Do not inspect or mutate the local kanban board.
- Stop immediately and instruct the user to use `godot-workflow:finalize-ticket`.
- Explain that the repo now uses Jira as the ticket source of truth and `merge-it` is no longer authoritative.
